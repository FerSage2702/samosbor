# Самосбор — карточная игра

Веб-приложение на PHP: карта с развилками, бои с врагами, карты оружия и лечения.

## Запуск

Нужен PHP 7.4+ с расширением session (обычно включено).

```bash
cd C:\Users\kosti\Desktop\samosbor
php -S localhost:8000
```

Открой в браузере: **http://localhost:8000**

Либо положи проект в каталог веб-сервера (XAMPP, OpenServer и т.п.) и открой соответствующий URL.

## Как играть

1. **Карта** — выбирай, куда идти (узлы: враги, отдых, магазин). Есть развилки.
2. **Враги** — на клетке с врагами появляется 1–3 противника (зависит от числа побед). У каждого своя картинка; бывают обычные и агрессивные (больше HP и урона).
3. **Бой** — выбираешь карту. Для оружия по одной цели (нож, топор, гвоздомет, ракетница) нужно выбрать врага. Сгушка бьёт по всем сразу.
4. **Карты** не тратятся: кулак, нож, хлеб с начала; остальные открываются при посещении локаций и магазинов. Хлеб тратится по штуке за использование; на отдыхе можно подобрать ещё (макс. 3).

## Карты

| Карта              | Эффект |
|--------------------|--------|
| Кулак              | 3–8 урона по цели |
| Нож                | 12–18 урона по цели |
| Топор              | 18–28 урона по цели, шанс кровотечения |
| Сгушка             | 8–14 урона по **всем** врагам |
| Гвоздомет          | 3 выстрела по 6–10 урона в одну цель |
| Сигнальная ракета  | 15–22 по цели + 5 сплеш по остальным + огонь на 3 хода (4 урона/ход) |
| Аптечка            | +30 HP |
| Буханка хлеба      | +10 HP (тратится, пополняется на отдыхе/в магазине) |

## Файлы

- `index.php` — точка входа, отображение карты/боя и приём действий.
- `GameEngine.php` — логика игры (карта, бой, прогресс, разблокировка карт).
- `config.php` — параметры карт, врагов, карты и пороги прогресса.
- `assets/css/style.css` — стили.
- `assets/js/game.js` — отправка ходов без перезагрузки (где нужно).
- `assets/images/enemies/` — изображения врагов (SVG).

Карты открываются по мере прохождения узлов и магазинов (см. `unlock_locations` в `config.php`). Пороги числа врагов: 0–1 побед — 1 враг, 2+ — 2 врага, 5+ — 3 врага.
